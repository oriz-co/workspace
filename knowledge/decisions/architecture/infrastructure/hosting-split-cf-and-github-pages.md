Cloudflare Pages = apps, PWAs, interactive websites (the surfaces users *use*). GitHub Pages = static marketing/landing pages for software (CLIs, browser extensions, MCP servers, npm-pkg docs). Both platforms' ToS permit donation buttons (BuyMeACoffee/Sponsors/UPI). Reverses earlier "everything on GitHub Pages" call. Refines [`cloudflare-pages-apps-only`](./cloudflare-pages-apps-only.md). Locked 2026-06-25.
